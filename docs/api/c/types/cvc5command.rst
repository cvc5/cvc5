Cvc5Command
===========

Encapsulation of a command.
Commands are constructed by the :cpp:type:`input parser <Cvc5InputParser>` and
can be invoked on the :cpp:type:`solver <Cvc5>` and :cpp:type:`symbol manager
<Cvc5SymbolManager>`.
