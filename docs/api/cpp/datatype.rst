Datatype
========

This class represents a datatype. A :cpp:class:`cvc5::Datatype` is encapsulated
by a datatype :cpp:class:`Sort <cvc5::Sort>` and can be retrieved from a
datatype sort via :cpp:func:`cvc5::Sort::getDatatype()`.
